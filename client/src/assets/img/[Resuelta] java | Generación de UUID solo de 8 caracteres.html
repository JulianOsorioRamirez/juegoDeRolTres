Quiero generar UUID solo de 8 caracteres, ¿es posible?

No es posible ya que un UUID es un número de 16 bytes por definición. Pero, por supuesto, puede generar cadenas únicas de 8 caracteres de largo (consulte las otras respuestas).

También tenga cuidado con la generación de UUID más largos y la subcadena, ya que algunas partes del ID pueden contener bytes fijos (por ejemplo, este es el caso de los UUID MAC, DCE y MD5).

sanghoon2 Puntos 79

¿Que tal este? En realidad, este código devuelve un máximo de 13 caracteres, pero es más corto que el UUID.

 import java.nio.ByteBuffer;
import java.util.UUID;

/**
 * Generate short UUID (13 characters)
 * 
 * @return short UUID
 */
public static String shortUUID() {
  UUID uuid = UUID.randomUUID();
  long l = ByteBuffer.wrap(uuid.toString().getBytes()).getLong();
  return Long.toString(l, Character.MAX_RADIX);
}
